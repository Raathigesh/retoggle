### Inspector

The `<Inspector />` component shows all the active toggles. This can be placed anywhere in your component tree.

#### Usage

```
import {Inspector} from 'retoggle';
```

<Playground>
    <Inspector usePortal={false}/>
</Playground>

 > Change `usePortal` to `true` to see the difference
