### useColorKnob(label, value)

A color picker which let you pic a color

- `label` - A label for the color picker
- `value` - A color (string or hex value) as initial value


#### Usage

```javascript
import { useColorKnob } from "retoggle";
```

<Playground>
{() => {
    const [color, setColor] = useColorKnob("Color", "green");

    return (
        <React.Fragment>
            <Inspector usePortal={false} />
            {color}
        </React.Fragment>
    )
  }}
</Playground>
